## Interactive page types (catalog)

Reusable interaction patterns. Build each as a generic engine once, reuse across subjects — same instinct as `atom-3d.js` (one component, six element presets).

| Code | Pattern | Existing precedent | Best for |
|---|---|---|---|
| DIAGRAM | Labelled diagram, click part → reveal info panel | detail panel in `periodic-table.dc.html` | anatomy, cell organelles, atom structure, circuit symbols |
| MODEL3D | Rotatable 3D model, drag to orbit | `atom-3d.js` | atoms, molecules, orbits, crystal lattices |
| SIM | Slider(s) drive a live visual/output | temp slider → state colour in periodic table | rates of reaction, wave frequency, pendulum, population models |
| BUILDER | Drag-drop construction task, checked against a target | none yet | circuit building, balancing equations, food webs, Punnett squares, molecule building |
| ANIM | Step-through or play/loop animation, scrubbable | planned: photon-jump animation (electron-shells backlog) | mitosis stages, digestion, plate boundary formation, immune response |
| COMPARE | Before/after or side-by-side slider | none yet | heating curves, erosion over time, evolution before/after |
| GRAPH | Live plotting tool, user changes input → curve redraws | none yet | motion graphs, titration curves, growth curves, energy profiles |
| QUIZ | Timed/streak quiz, multiple modes | quiz mode in `periodic-table.dc.html` (LV.1/LV.2, score/streak) | any topic — recall/name/classify |
| LAB | Virtual experiment bench, safe consequence-free | none yet | titrations, electrolysis, circuit testing, dissection alternative |
| CALC | Step-revealed worked-example solver | none yet | mole calculations, SUVAT, Ohm's law, Punnett square solver, half-life |
| TIMELINE | Scrubbable timeline, zoomable | none yet | geological time, universe history, element discovery history |
| MAP | To-scale spatial explorer, pan/zoom | none yet (backlog: solar-system-to-scale) | solar system distances, plate boundaries, biomes |
| SORT | Drag-to-classify game | none yet | rock ID, organism classification keys, element property sorting |
| SANDBOX | Fully open-ended free play, no target/score | none yet | circuit sandbox, ecosystem designer, mixing chemicals freely |
| ARTICLE | Field-Guide long-form text, embeds any pattern above | `electron-shells.dc.html` | concept explainers that need prose + one embedded widget |

**Build priority** (covers the most topic-clusters per engine built):
1. DIAGRAM engine (generalize the detail-panel pattern) — anatomy, cells, atoms, circuits all reuse it
2. SIM engine (generic slider-in → visual-out) — biggest win, used across every subject
3. BUILDER (drag-drop + target-check) — circuits, equations, food webs, genetics
4. QUIZ engine generalized from periodic-table (already 80% reusable — extract into shared component)
5. GRAPH, CALC, ANIM, LAB, MAP, TIMELINE, SORT, COMPARE, SANDBOX — build as needed per topic

## Topic → interaction mapping

### Chemistry
- States of matter / particle model (L1-L3): SIM (particle box vs temp) + COMPARE (heating curve) + QUIZ
- Atoms, elements, periodic table (L1-L3): DIAGRAM + MODEL3D (reuse `atom-3d`) + QUIZ — already live
- Bonding: ionic/covalent/metallic (L3-L5): DIAGRAM (electron dot, lattice) + MODEL3D + BUILDER (drag atoms/bonds to build a molecule)
- Chemical reactions & equations (L2-L3): BUILDER (balance equation via drag coefficients) + LAB (mix reagents, see product) + QUIZ
- Rates of reaction (L3): SIM (temp/conc/surface-area/catalyst sliders → rate GRAPH)
- Energetics / enthalpy (L3-L5): GRAPH (interactive energy profile diagram) + CALC
- Acids, bases, pH (L1-L3): SIM (pH slider + indicator colour) + LAB (virtual titration)
- Electrolysis (L3): LAB + ANIM (ion migration)
- Organic chemistry (L3-L5): MODEL3D (ball-and-stick molecule builder) + BUILDER (name/build molecule) + SORT (functional groups)
- Moles / stoichiometry (L3-L5): CALC (step-revealed mole calculator, practice generator)
- Equilibria (L4-L5): SIM (Le Chatelier shift visualizer) + GRAPH
- Electrochemistry (L5): LAB (build a cell, predict voltage) + CALC
- Kinetics (L5): GRAPH (determine order from data) + SIM (Arrhenius plot)
- Spectroscopy (L5): SIM (click spectrum peaks → structure clues)
- Earth's atmosphere / resources: MAP + TIMELINE (climate change) + GRAPH

### Physics
- Forces & motion (L1-L4): BUILDER (free-body diagram) + SIM (tug of war) + GRAPH (draw-and-check motion graphs)
- Simple machines (L1): SANDBOX (lever/pulley — drag load, see effort)
- Light, sound, waves (L1-L4): SIM (frequency/amplitude sliders) + ANIM (ray-tracer reflection/refraction)
- Electricity / circuits (L1-L4): BUILDER (drag-drop circuit, live current animation) + CALC (Ohm's law solver) — high-value shared engine, reused constantly
- Magnetism / electromagnets (L2-L4): SIM (field-line visualizer, coil sandbox)
- Energy & power (L2-L4): SIM (Sankey-style transfer diagram) + CALC
- Momentum & collisions (L3-L5): SANDBOX (set masses/velocities, watch elastic/inelastic) + GRAPH
- Radioactivity / nuclear (L3-L5): SIM (half-life decay, atoms decay on click) + TIMELINE
- Space / astrophysics (L2-L5): MAP (solar system to scale) + MODEL3D (orbit sim) + ANIM (star life-cycle stepper)
- Kinematics / SHM (L4-L5): SANDBOX (pendulum/spring) + GRAPH (live output)
- Fields — grav/electric/magnetic (L5): SIM (place charges/masses, see field lines) + MODEL3D
- Quantum (L5): ANIM (photoelectric effect interactive)

### Biology
- Cells (L1-L5): DIAGRAM (click organelle → function) + MODEL3D (3D cell explorer) + QUIZ
- Body systems (L1-L4): DIAGRAM (interactive body map) + ANIM (digestion/circulation flow)
- Genetics (L2-L5): BUILDER (Punnett square generator) + CALC
- Photosynthesis / respiration (L2-L5): SIM (light/CO2 sliders → rate GRAPH) + ANIM (pathway)
- Ecosystems / food webs (L2-L5): BUILDER (build food web) + SIM (predict population shifts) + SORT
- Evolution / natural selection (L2-L5): SIM (selection-pressure simulator over generations) + TIMELINE
