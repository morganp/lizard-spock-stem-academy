// FIELD GUIDE 04 — LEVEL L1–L2

Elements, compounds & mixtures

Everything around you is built from these three patterns. Learn to spot the difference, then sort a few yourself.
// FIG. 01 — ELEMENT PARTICLE DIAGRAM
Element
One type of atom only. Can't be broken into anything simpler. Every square on the periodic table is one.
e.g. OXYGEN, GOLD, CARBON
// FIG. 02 — COMPOUND PARTICLE DIAGRAM
Compound
Two or more elements chemically bonded in a fixed ratio. Only a reaction can pull it back apart.
e.g. WATER (H₂O), SALT (NaCl)
// FIG. 03 — MIXTURE PARTICLE DIAGRAM
Mixture
Substances jumbled together, not bonded — no fixed ratio. Easy to pull back apart with physical methods.
e.g. AIR, SALAD, SEAWATER
