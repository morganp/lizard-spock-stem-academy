// TISSUES, ORGANS & ENZYMES — LEVEL L3

Cells organise into systems — enzymes make them work

Life is built in layers: cells → tissues → organs → organ systems. And almost every reaction inside them is sped up by an enzyme — a protein with a precisely shaped active site. Run the enzyme bench below to see the lock-and-key fit, and what happens when you heat it too far or change the pH.

// GRAPH 01 — HOW {{ enzName }} ACTIVITY CHANGES WITH TEMPERATURE
Activity rises with temperature (more collisions) up to the optimum, then falls off a cliff: past ~40 °C the enzyme's active site changes shape — it denatures — and the substrate no longer fits. Denaturing is permanent.
